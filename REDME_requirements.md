# Arduino Traffic Light System (North–South)

This project demonstrates a real-time traffic light system using Arduino.

## Components Used
- Arduino Uno
- Breadboard
- LEDs (Red, Yellow, Green)
- Resistors (220Ω)
- Jumper wires

## Working
The Arduino controls traffic lights for North and South directions using time delays.
